## What is typemap?

**typemap** is a runtime type evaluation library for Python 3.14+ that implements the type manipulation operators from [PEP 827](https://peps.python.org/pep-0827/).

This project is a packaged version of the original [vercel/python-typemap](https://github.com/vercel/python-typemap) implementation, distributed via PyPI for easy installation.

## Installation

```bash
pip install typemap
```

## Quick Start

```python
from typemap import eval_typing
import typemap_extensions as tm

class User:
    name: str
    age: int

# Evaluate a type at runtime
result = eval_typing(tm.KeyOf[User])
print(result)  # tuple[Literal["name"], Literal["age"]]
```
