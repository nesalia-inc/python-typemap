## Core Operators

typemap implements the core type operators from PEP 827.

### Member

Access type members by name:

```python
import typemap_extensions as tm
from typing import Literal

class User:
    name: str
    age: int

# Get member named "name"
member = tm.Member[Literal["name"], str]
```

### Attrs

Get all attribute members of a class:

```python
import typemap_extensions as tm

class User:
    name: str
    age: int
    email: str

# Get all attributes
attrs = tm.Attrs[User]
# Returns tuple of Member types for each attribute
```

### Iter

Iterate over type contents in comprehensions:

```python
import typemap_extensions as tm
from typing import Literal

class User:
    name: str
    age: int

# Iterate over attributes
for attr in tm.Iter[tm.Attrs[User]]:
    print(attr.name)  # "name", "age"
```

### NewProtocol

Create a new structural protocol:

```python
import typemap_extensions as tm
from typing import Literal

# Create a protocol with specific members
Protocol = tm.NewProtocol[
    tm.Member[Literal["name"], str],
    tm.Member[Literal["age"], int],
]
```

### UpdateClass

Generate class modifications:

```python
import typemap_extensions as tm
from typing import Literal, Callable, Self

type InitFn[T] = tm.Member[
    Literal["__init__"],
    Callable[[tm.Param[Literal["self"], Self], *params], None],
]

class Model:
    def __init_subclass__(cls) -> tm.UpdateClass[InitFn[cls]]:
        super().__init_subclass__()
```

## Boolean Operators

### IsAssignable

Check if a type is assignable to another:

```python
import typemap_extensions as tm
from typemap import eval_typing

result = eval_typing(tm.IsAssignable[int, float])
# Returns: True

result = eval_typing(tm.IsAssignable[str, int])
# Returns: False
```

### IsEquivalent

Check if two types are equivalent:

```python
import typemap_extensions as tm
from typemap import eval_typing

result = eval_typing(tm.IsEquivalent[int, int])
# Returns: True
```

## String Operators

These are potential future extensions from PEP 827:

### Template

Build template literal strings:

```python
import typemap_extensions as tm
from typing import Literal

type Route = tm.Template["api/", Literal["users"], "/", Literal["id"]]
# Returns: Literal["api/users/id"]
```

### KeyOf

Get all member names as tuple of Literals:

```python
import typemap_extensions as tm
from typemap import eval_typing

class User:
    name: str
    age: int
    email: str

keys = eval_typing(tm.KeyOf[User])
# Returns: tuple[Literal["name"], Literal["age"], Literal["email"]]
```
